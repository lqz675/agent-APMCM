# 更新日志

## 2026-06-11: Reranker 架构升级

### 变更摘要
新增多 Provider Reranker 支持，Jina 为首选，Qwen 阿里云为自动降级备选。

### 涉及文件
| 文件 | 变更 |
|---|---|
| `.env` | 新增 JINA_API_KEY、JINA_RERANK_MODEL、QWEN_API_KEY、QWEN_BASE_URL、QWEN_RERANK_MODEL |
| `.env.example` | 同步新增上述配置项 |
| `app/model.py` | 新增 `rerank()` 函数 + `_rerank_qwen()` 内部函数，支持 Jina/Cohere/Qwen 三 Provider |

### 降级逻辑
```
rerank(query, docs)
  ├── provider="qwen"  → _rerank_qwen() → text-embedding-v4 余弦相似度
  ├── provider="cohere" → Cohere /v2/rerank API
  ├── 默认: Jina /v1/rerank API
  └── Jina 失败 → 自动降级 _rerank_qwen()
```

### Provider 对比
| Provider | 模型 | 方式 | 状态 |
|---|---|---|---|
| Jina | jina-reranker-v2-base-multilingual | 专用 rerank API | 首选（需开 VPN） |
| Qwen | text-embedding-v4 | embedding + 余弦相似度 | 备选（国内直连） |
| Cohere | rerank-multilingual-v3.0 | 专用 rerank API | 未配置 |

### 配置说明
- 国内环境（当前）：QWEN 使用 `https://dashscope.aliyuncs.com/...`
- VPN 环境：QWEN 使用 `https://dashscope-us.aliyuncs.com/...`（.env 中已注释）
- 比赛开始时将国内 QWEN key 注释掉，启用 VPN 版本
