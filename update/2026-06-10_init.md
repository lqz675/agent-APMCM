# 更新日志

## 2026-06-10: Agent 初始架构建立

### 变更摘要
从零搭建完整的数学建模 Agent，包含 9 阶段工作流、RAG 检索引擎、多 Skill 集成。

### 涉及文件
| 文件 | 说明 |
|---|---|
| `app/main.py` | Streamlit 9 阶段工作流界面 |
| `app/rag.py` | FAISS 向量检索引擎，3 索引库 |
| `app/model.py` | DeepSeek Chat + NVIDIA Embedding 双 API |
| `app/prompts.py` | 8 个阶段 Prompt 模板 |
| `app/utils.py` | PDF 解析（支持子文件夹递归）、缓存 |
| `app/skills_bridge.py` | 6 个外部 Skill 桥接 |
| `app/workflow_logger.py` | 双格式工作流日志（JSON + MD） |
| `skills/` | 6 个克隆仓库（ragflow、pressure-test、mattpocock/skills、waza、scipilot-figure、gpt_academic） |
| `dataset/{problems,papers,references}/` | 数据目录 |
| `.env` / `.env.example` | API 配置 |
| `requirements.txt` | Python 依赖 |
| `introduce/architecture.md` | 架构文档 |
| `introduce/user-guide.md` | 操作手册 |

### 技术选型
- **LLM**: DeepSeek-chat（后期可换 ChatGPT 5.5）
- **Embedding**: NVIDIA nv-embed-v1（API 调用，不本地部署）
- **RAG**: 借鉴 RAGFlow 架构，自建 FAISS IndexFlatL2
- **UI**: Streamlit

### 工作流
1. 上传赛题 → 2. 选题分析 → 3. 建模方案 → 4. 压力测试 → 5. 用户对齐 → 6. 代码生成 → 7. 图表生成 → 8. 论文初稿 → 9. 论文润色
