# 对话记忆目录

存放每次会话的实时对话记录，按阶段分文件保存。

## 目录结构

memory/
└── {session_id}/
    ├── stage_01_input_20260612_143000.md
    ├── stage_02_topic_selection_20260612_143210.md
    └── ...

## 文件命名规则

stage_{阶段序号:02d}_{阶段名}_{YYYYMMDD_HHMMSS}.md

## 用途

- 完整保留与 agent 的对话过程
- 阶段结束后可手动上传给网页版 AI 提供上下文
- 可作为后续分析或复盘材料

> 文件在每条消息发送后立即写入（实时 flush）
